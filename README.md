# Слот-вотчер для ПроДокторов

Следит за расписанием выбранного врача на [prodoctorov.ru](https://prodoctorov.ru)
и присылает вам сообщение в Telegram, как только появляется свободное время
для записи (например, кто-то отменил приём). Плюс раз в день — сводка,
что бот жив и что с расписанием.

Работает даже когда у клиники отключена онлайн-запись: расписание всё равно
видно, а записываться в этом случае нужно по телефону — бот присылает
телефоны клиник прямо в уведомлении.

## Что понадобится

- Компьютер (Mac, Windows или Linux), который будет включён и **не будет
  засыпать**, пока идёт слежение.
- Telegram.
- Ссылка на страницу вашего врача, например
  `https://prodoctorov.ru/zelenograd/vrach/304702-tonyan/`.

## Установка с помощью ИИ (рекомендуется)

Не нужно разбираться в программировании. Откройте Claude
([claude.ai](https://claude.ai)) или ChatGPT и отправьте ему это сообщение
целиком, подставив свою ссылку на врача:

```
Помоги мне установить и запустить программу-наблюдатель за записью к врачу.

Репозиторий: https://github.com/ВАШ_ЛОГИН/prodoctorov-slot-watcher
Врач, за которым надо следить: <ВСТАВЬТЕ ССЫЛКУ НА ВРАЧА>

Сделай для меня по шагам, объясняя каждый шаг простыми словами:
1. Проверь, установлен ли Go (нужна версия 1.23 или новее); если нет — помоги установить.
2. Скачай репозиторий (git clone) и собери программу: go build -o watcher .
3. Запусти ./watcher и помоги мне ответить на вопросы мастера настройки:
   он попросит ссылку на врача, токен Telegram-бота (поможешь создать через
   @BotFather) и пару настроек.
4. Настрой, чтобы программа работала постоянно: не закрывалась вместе с
   терминалом и чтобы компьютер не засыпал (на Mac — например, caffeinate
   или настройки энергосбережения).

Я не программист — команды выполняй сам или диктуй мне по одной.
```

ИИ проведёт вас за руку: установит всё нужное, соберёт программу и поможет
пройти настройку.

## Установка вручную

1. Установите [Go](https://go.dev/dl/) 1.23+.
2. `git clone <адрес репозитория> && cd slot-watcher`
3. `go build -o watcher .`
4. `./watcher` — при первом запуске мастер задаст 5 вопросов:
   ссылка на врача → токен бота (создаётся у @BotFather) → сообщение боту
   (chat_id определится сам) → как часто проверять (по умолчанию раз в
   20 минут) → во сколько присылать ежедневную сводку (по умолчанию 09:00).

Настройки сохраняются в `config.json` рядом с программой. Чтобы настроить
заново — удалите `config.json` и запустите ещё раз. Чтобы поменять одну
настройку — отредактируйте `config.json` и перезапустите программу.

## Частые вопросы

**Хочу следить за двумя врачами.** Сделайте вторую копию папки со своим
`config.json` и запустите вторую копию программы.

**Почему нельзя проверять чаще, чем раз в 10 минут?** API неофициальный;
щадящий режим — гарантия, что слежение не создаёт нагрузку на сайт и
не будет заблокировано.

**Пришло уведомление — что делать?** Звонить в клинику по телефону из
уведомления (если у клиники отключена онлайн-запись, кнопки на сайте не
будет) или записываться на сайте, если кнопка есть. Слоты разбирают быстро.

**Компьютер уснул — бот молчал?** Да, программа работает только пока
компьютер не спит. Попросите ИИ из шага установки настроить режим
«не засыпать» — или запускайте на компьютере, который всегда включён.

## Дисклеймер

Проект использует неофициальный API prodoctorov.ru — формат может измениться
в любой момент, тогда бот сообщит об ошибках и потребуется обновление.
Проект не связан с ПроДокторов. Используйте в личных целях.

# ProDoctorov slot-watcher — заметки

## Ключевой тейк (подтверждено 2026-07-01)

**API `slots_bulk` доступен всегда и не зависит от подписки клиники.**

- Клиника на тарифе **«Лайт»** → на публичной странице врача
  (`/moskva/vrach/975987-hafez/`) **нет виджета «Выбрать дату и время»**,
  и слот **нельзя забронировать онлайн** — только по телефону.
- При этом backend-API
  `POST https://prodoctorov.ru/ajax/schedule/slots_bulk/`
  **возвращает реальное расписание врача, даже когда виджет выключен.**
- Доказательство: 2026-07-01 виджет был выключен, а API вернул свободный слот
  `lpu 70382 (корп. 1501) → 2026-07-30 08:00 free:true`.

**Вывод:** тариф/подписка управляют только кнопкой записи на сайте, а НЕ доступностью
API. Раннее предположение «интеграция включается/выключается вместе с подпиской» —
**неверно**. API — надёжный сигнал, который работает постоянно и ловит освободившиеся
по отмене слоты, невидимые на сайте.

## Практика
- Записываться на пойманный слот — **по телефону** клиники (онлайн-кнопки не будет,
  пока клиника не вернёт интеграцию).
- Нужен свежий `csrftoken` (один GET перед запросом) — watcher делает сам.
- Неофициальный API: формат могут поменять; опрос раз в 20 мин — щадящий режим.
- Сигнал работает только пока сервер онлайн (сейчас — Mac, не во сне).

## Архитектура публичной версии (2026-07-13)

Конфиг вынесен в config.json (мастер первого запуска), врач и клиники
автоопределяются со страницы врача (блок doctorsLpus + :lpu-address-list),
heartbeat заменён ежедневной сводкой. Дизайн:
docs/superpowers/specs/2026-07-13-public-slot-watcher-design.md
